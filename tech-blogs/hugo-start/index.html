使用githubPage+hugo无成本建立博客

Nov 30, 2024
2 minute read

0 前言

环境：ubuntu22.04

  • GitHub Pages 是一个静态站点托管服务，可以直接将页面托管于 GitHub 库或仓库 (repository) 中。

  • Hugo 是一个用 Go 语言编写的静态站点生成器，其优势为速度快。（还有hexo、jekyll）

1 安装hugo 

sudo apt-get install hugo
# 或
sudo snap install hugo

# 测试安装
hugo version

2 本地创建 

# 创建
cd ~
hugo new site 【project-name】
cd ~/project-name

# 安装主题. 可以在github搜索hugo theme. 找适合自己的, 这里选用Stack.
cd ~/project-name/theme
git submodule add https://github.com/CaiJimmy/hugo-theme-stack theme/Stack

修改配置文件hugo.toml

baseURL = 'https://【github用户名】.github.io/'
languageCode = 'zh-cn'
title = 'MY Site'
theme = 'Stack'

# 本地预览
hugo server

# 网页输入
http://localhost:1313/

# 构建Hugo网站，将静态站点保存到public下
hugo

3 上传到github

这里选择使用两个仓库

  • 一个仓库储存整个代码, 设置为private. 并把/public文件夹添加到.gitignore

  • 第二个仓库储存public文件夹下的静态代码. 仓库名需要为 github用户名.github.io

  • 在自己的github下新建repo. 仓库名随意，应为【project-name】

cd ~/project-name
git init

# 添加.gitignore
cd ~/project-name
touch .gitignore
echo "/public" > .gitignore

git add .
git commit -m "repo init"
git branch -M main
git remote add origin git@github.com:【github用户名】/【仓库名】.git
git push -u origin main
  • 第二个仓库
cd ~/project-name/public/
git init
git add .
git commit -m "page first commit"
git branch -M main
git remote add origin git@github.com:【github用户名】/【github用户名.github.io】.git
git push -u origin main

4 快捷方式 

alias inblog='cd /home/zrf/rfblogs_src/'
alias blogpush='inblog && cd public && git add . && git commit -m "update" && git push && cd ../'

5 使用Utterances配置评论

Utterances 是一个基于 GitHub Issues 的轻量级评论系统。它的主要优点是：

  • 无需额外账户，直接使用 GitHub。
  • 评论存储在 GitHub Issues 中，不需要额外的服务器。
  • 免费、开源，并且加载快速。

Utterances官网：https://utteranc.es/

  1. 为项目博客安装Utterances： https://github.com/apps/utterances
  2. 设置配置 hugo.toml 中添加
[Params.Comments]
    enabled = true
    provider = "utterances"

[Params.Comments.utterances]
    repo = "UserName/repo"
    issueTerm = "pathname"
    label = "comment"
    theme = "github-light"
    crossorigin = "anonymous"
    async = true

6 配置头像

图像路径：‘assets/img/avatar.png’

    src = "img/avatar.png"
    local = true
